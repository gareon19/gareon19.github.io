Created Applications

ShoppingListGenerator

The ShoppingListGenerator is an application that facilitates the weekly shopping. The user can create a weekly menu plan and receives the list of ingredients to prepare these menus. The available menu list can be found and edited in the menus.xml file. Each menu has a name and contains multiple ingredients. Each ingredient has a name an amount and a location at which it can be found in the store. The application is currently only available in german. Feel free to contact me for questions.
